# Sports Data Organization Project

This project focuses on cleaning, organizing, and restructuring raw sports team data across multiple worksheets. The goal was to transform a single cluttered dataset into organized, structured, and easy-to-read Excel workbooks.

---

## Objectives

- Separate raw combined data into two structured team workbooks  
- Create organized worksheets:
  - Match Results  
  - Player Data  
  - Team Stats  
- Format headers and apply consistent layout  
- Align and clean inconsistent values  
- Navigate efficiently using Excel shortcuts  
- Export player data to PDF  
- Use Microsoft Copilot to automate repetitive tasks  

---

## Files in This Project

- `Team One.xlsx` – Cleaned and organized workbook for Team One  
- `Team Two.xlsx` – Cleaned and organized workbook for Team Two  
- `raw_dataset.xlsx` – Original dataset before processing  
- `Team One Player Data.pdf` – PDF export  
- `Team Two Player Data.pdf` – PDF export  
- `images/` – Folder containing screenshots documenting progress  

---

## Process Summary

1. Loaded the provided dataset into Excel  
2. Created two new workbooks (one for each team)  
3. Built structured worksheets and labeled them clearly  
4. Copied and aligned player data from the raw sheet  
5. Cleaned and standardized formatting, layout, and values  
6. Applied styling to headers (bold, background color, alignment)  
7. Exported the Player Data worksheets to PDF  
8. Tested automated formatting through Microsoft Copilot
