# Monthly Household Budget Project  

This project builds a reusable monthly household budget workbook in Excel.

**Skills practiced:**  

- Creating and organizing multiple worksheets (Budget, Detailed Tracker, Budget Overview)
- Entering and cleaning income and expense data
- Converting text values to numbers and dates
- Applying currency and date formats
- Using formulas for summary calculations (e.g., `SUMIF, budget vs. actual, differences)
- Linking sheets to build a small "dashboard" style overview
- Exporting the budget summary to PDF
